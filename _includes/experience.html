Researches

Visual Question Answering

2016 - Present
HKPolyU, Hong Kong

A system takes in as input an image and natural language question, outputs an answer to the query based on image contents. Inspired by object proposal approaches, try to extract finer-grained features with edge box for image representation. Due to variety of question types, try to apply question factorization to train question-specific model for better overall performance.

Automatic Fabric Defect Detection

2015 - Present
HKPolyU, Hong Kong

A system can automatically detect fabric defects with the technology of image processing, can also outputs reports.Developing algorithm that inspects the defects shown on fabric constantly woven with matlab and opencv.Designing and implementing the software, which integrates hardware parts, e.g. multiple cameras, shaft encoder.

Fog Detection and Alarm System

2015
Nanjing, China

Detection device captures image with proper intervals and puts it to process to evaluate fog situation, then uploads results and typical image to Server for future process. Designed a practical approach for fog detection based on multiple experiments. Developed the algorithm that receives an image and output the ranked severeness of the fog. Implemented most part of the project, wrote both the Terminal (an app) and Server (Web).

Intelligent Vehicle for Rut Detection

2014 - 2015
Nanjing, China

A real-time system that captures the image of the road and uses image-processing technology to automatically calculate the rut. Proposed an algorithm extracting the line-structured feature to represent the rut, and mimicking the real measurement approach to calculate the depth of the rut. Integrated data capturing, processing and rearranging parts together, which can show the results within low latency.

Optical Character Recognition (OCR)

2014
Nanjing, China

Wrote an android app that captures or loads an image and outputs the characters (Both English and Chinese). Made improvements to the preprocessing part that enhanced the recognition rate up to 96%.

Road Detection for Unmanned Vehicle

2014
Nanjing, China

Studied the work principle of Velodyne HDL-64E Lidar, and translated the raw 3D data from the lidar into a grid map. Clustered the obstacles on the grid map and extracted feasible paths for navigation.
